We can see here the performance of different instance types with same limits is almost the same. Also check the performance of those limited instances to the actual instance performance.

Conclusions:

  • Different types of instances with same limits gives almost same performance with one condition the instance has the necessary resources
  • Performance of limited Instance is again almost same as the performance of a virtual machine with the same amount of resources. There will be slight drop of performance towards the peak due to the fact that VM is using some resources undearneath and not all resources are available as compared to the limited instance.
  • Resource Utilization of different pods/replicas on the same machine is exactly the same.
